# Architecture Decision Records (ADRs)

This directory contains the Architecture Decision Records (ADRs) for the Sentio Systems (Orbis + Sentio) project.

## What is an ADR?
An ADR is a document that captures an important architectural decision made along with its context and consequences. We use ADRs to:
* Keep a history of decisions.
* Onboard new team members.
* Facilitate agreement within the team.

## Template
We use the [MADR](https://adr.github.io/madr/) format.
