# ADR-0005: React + TypeScript + Vite for Frontend

## Status
Accepted

## Context
The `sentio-web` dashboard needs to display live data, render complex maps (Leaflet), and visualize charts. We need a modern, component-based library with a fast build tool and type safety to prevent runtime errors.

## Decision
We will use **React** with **TypeScript** and **Vite** as the build tool.

## Consequences
**Positive:**
* **React:** Huge ecosystem for UI components (shadcn/ui, recharts).
* **TypeScript:** Catches errors at compile time, improving code quality.
* **Vite:** Significantly faster development server and build times compared to Webpack/CRA.

**Negative:**
* Learning curve for strict TypeScript typing.
