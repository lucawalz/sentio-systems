# ADR-0011: Hailo AI Accelerator for Edge Inference

## Status
Accepted

## Context
The Orbis Hub needs to analyze camera streams to detect wildlife. Sending every video frame to the cloud is bandwidth-prohibitive. 
The Raspberry Pi CPU is not powerful enough for high-FPS object detection.

## Decision
We will use the **Hailo AI Accelerator (NPU)** on the Edge device.

## Consequences
**Positive:**
* Enables real-time inference on the edge.
* Significant reduction in bandwidth (only send images when objects are detected).

**Negative:**
* Hardware dependency (code cannot run on standard x86 dev machines without the dongle).
* Development complexity: Models must be compiled to `.hef` format.
