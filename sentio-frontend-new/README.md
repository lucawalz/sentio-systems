# sentio-frontend-new

## Overview
This is the new frontend for Sentio Systems, replacing the legacy `sentio-web`.

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```

## Relationship to sentio-web
This project supersedes `sentio-web` and contains updated UI and features. Use this for all new development.
